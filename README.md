## 💻 CLI Usage

Launch from command line with optional flags:

```bash
AuthenticSoundware.exe --help
AuthenticSoundware.exe --profile gaming
AuthenticSoundware.exe --minimized --profile voice
```

| Flag | Description |
|------|-------------|
| `--profile <name>` | Load a specific profile on startup |
| `--minimized` | Start minimized to system tray |
| `--reset` | Reset all settings to defaults |
| `--version` | Display version information |
